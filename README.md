# ⬡ PChat — Secure & Compliant Messaging

PChat is a modern web-based messaging application developed for the Indian market. It prioritizes user privacy while maintaining a robust architecture for legal compliance and administrative management.

## 🚀 Key Features
* **Private Messaging:** Secure user-to-user communication.
* **Compliance Ready:** Designed with a modular administrative backend that allows for data retrieval in authorized legal or regulatory scenarios.
* **Modern UI:** Responsive, dark-themed interface built with glassmorphism aesthetics.
* **Sign-In/Sign-Up:** Complete user lifecycle management with secure authentication.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Features:** Responsive Layout, Authentication Logic, Admin Dashboard (WIP)
* **Design Philosophy:** "Made for India" — Clean, fast, and accessible.

## ⚖️ Privacy & Compliance Policy
PChat is built on the principle of "Privacy with Accountability." While user messages are private, the platform is designed to comply with local regulations, allowing administrative access strictly for legal cases and authorized requests.
